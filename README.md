# MusicNow
##### Download music with album art and details.

### Dependencies

##### Mac

```sh
$ brew install libav
```


##### Ubuntu
```sh
$ sudo apt-get install libav-tools
```

### Installation

##### Python 2.7
```sh
$ pip install musicnow
```

##### Python 3.4
```sh
$ pip3 install musicnow
```




### How to use
```sh
$ musicnow
```
