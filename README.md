# Download-Music
##### Enter song name and download the song with album art and song details.

### Dependencies

##### Mac

```sh
$ brew install libav
```


##### Ubuntu
```sh
$ sudo apt-get install libav-tools
```

### Installation
```sh
$ sudo pip install download-music
```



### How to use

```sh
$ download-music

> Enter Song Name/Keywords : When the levee breaks

```
